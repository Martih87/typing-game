# ⌨️ Type It Right!

A typing practice game for kids with 500 Danish sentences. Built with TypeScript + Vite.

## Prerequisites

- [Node.js](https://nodejs.org/) (v18+)

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Opens on `http://localhost:5173` and is accessible on your local network.

## Production

```bash
npm run build
npm run preview
```

This builds and serves the app on your local network (default port 4173).
